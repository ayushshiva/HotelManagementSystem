# 🏨 Hotel Management System

![Java](https://img.shields.io/badge/Java-17+-orange)
![Swing](https://img.shields.io/badge/GUI-Java%20Swing-blue)
![Status](https://img.shields.io/badge/Status-Completed-success)
![Platform](https://img.shields.io/badge/Platform-Desktop-lightgrey)

A desktop-based **Hotel Management System** developed using **Java Swing** to simplify hotel operations through a user-friendly graphical interface. The application allows hotel staff to manage room bookings, customer records, food orders, laundry services, employee information, billing, and payments from a centralized dashboard.

The project demonstrates core Java concepts including Object-Oriented Programming (OOP), event-driven programming, file handling, collections, and GUI development.

# 📖 Overview

The Hotel Management System provides an easy-to-use desktop interface for handling common hotel operations.

The application includes modules for:

- Room Booking
- Customer Management
- Food Ordering
- Laundry Services
- Bill Generation
- Payment Calculation
- Employee Information
- Receipt Logging

---

# ✨ Features

## 🔐 Login System

- Secure login interface
- Username & password validation
- Password visibility toggle
- Remember Me option in UI

---

## 🏨 Room Booking

- Book available rooms
- Room category selection
- AC / Non-AC selection
- Automatic room availability management
- Customer details storage

---

## 🍽 Food Ordering

- Multiple food categories
- Price calculation
- Order summary
- Customer order recording

---

## 🧺 Laundry Service

- Cloth type selection
- Quantity selection
- Automatic bill calculation

---

## 💰 Billing System

- GST calculation
- Final bill generation
- Balance amount calculation after payment

---

## 👥 Customer Management

- Customer table
- View booked customers
- Delete customer records
- Automatically free rooms after deletion

---

## 👨‍💼 Employee Section

Displays hotel employee information including:

- Manager
- Receptionist
- Chef
- Cleaner

---

## 📝 Receipt Logging

Important hotel activities are automatically saved to a receipt log file for record keeping.

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ayushshiva/HotelManagementSystem.git
```

Open the project in your preferred Java IDE.

Compile all Java files:

```bash
javac *.java
```

Run the application:

```bash
java Main
```
